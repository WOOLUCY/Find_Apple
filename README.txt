# 게임 설치 및 실행 방법
1. setup.exe를 클릭해 실행한다.
2. 확인 버튼을 누른다.
3. Next 버튼을 누른다.
4. 약관에 동의 후, Next 버튼을 누른다.
5. 사용자명과 조직을 입력한다. 필수 입력 사항이 아니다.
6. Custom을 선택 후, Next 버튼을 누른다.
7. Change 버튼을 눌러 원하는 경로를 입력한 뒤, Next 버튼을 누른다.
8. Install 버튼을 누른다.
9. 설치가 완료되면, 입력했던 경로로 이동한다.
10. FindApple_Client 폴더 내의 Gameproject.exe를 눌러 게임을 실행할 수 있다.

# 기타 사항
- 서버를 위한 코드와 실행 파일은 FindApple_Server에 위치한다.
- 프로그램에 사용된 소스 파일은 FindApple_Source에 위치한다.
- 서버는 일정 IP에서만 작동한다.
- 설치가 된 이후에, setup.exe를 다시 실행하면 Remove를 통해 프로그램을 삭제할 수 있다.
